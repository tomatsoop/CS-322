# CS-322 - Intro to Software Engineering
Spring 2026

The class has a framework intergrating different backend development practices. The main template application calcuatets ACP brevet control open/close times. Main idea of this course it to understand application development and package distribution through Docker. Much of the code was given by the CS 322 department by Professor Micheal Young. Project 4 through 6 was implemented with AI pair programming. 

Project 1: Basic page server, learning how ports and sockets operate. Basic html serve and response.

Project 2: Introduction to Docker and Python Flask. Render HTML page from Docker container. 

Project 3: Introduction to Flask, AJAX, and JSON. Understanding and implementing dynamic content rendering.

Project 4: Base code for Brevets project. Understanding nose testing, conditional writing, and flask error handlers.

Project 5: Implementation of MongoDB into a saved data base for display. 

Project 6: Implementing REST API with MongoDB and multiple container impelmentation with Docker.
